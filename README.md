# Análisis de Ventas y Tendencias del Mercado de Videojuegos

Analicé datos históricos de ventas de videojuegos para identificar los factores asociados al éxito comercial de un juego.
El objetivo fue detectar patrones relacionados con plataformas, géneros, calificaciones de usuarios y preferencias regionales para generar insights útiles sobre el comportamiento del mercado.

Donde se aplicaron las técnicas siguientes:
 - Limpieza de datos
 - Análisis exploratorio (EDA)
 - Visualización de datos
 - Análisis estadístico
 - Evaluación de hipótesis

 Donde obtuvimos las siguientes conclusiones generales:
  - Las plataformas con mayor popularidad presentan ciclos de vida relativamente cortos, pero generan altos volúmenes de ventas durante su pico de adopción.
  - Los géneros Action y Shooter dominan el mercado global, especialmente en Norteamérica y Europa.
  - Las calificaciones de críticos muestran una relación más fuerte con las ventas que las calificaciones de usuarios.
  - Existen diferencias claras entre regiones: mientras ciertos géneros dominan en Occidente, en Japón los juegos de rol tienen un desempeño significativamente mejor.
